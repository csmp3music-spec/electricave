# Historical Boston Rapid Transit Pass

This pass adds three corridor families to the procedural network builder:
- Atlantic Avenue Elevated
- Washington Street Tunnel
- Cambridge-Dorchester Tunnel (historic Harvard-to-Ashmont sequence)

## Implementation notes

- Tremont / Park Street remains the older warm-tile, brick-headhouse streetcar subway treatment.
- Washington Street Tunnel uses offset side-platform station geometry and a tighter orange-tinted finish.
- Cambridge-Dorchester uses a broader heavy-rail treatment with more island-platform geometry, based on the historic Cambridge and Dorchester tunnel sequence.
- Atlantic Avenue Elevated is implemented as a separate steel-and-timber elevated structure rather than another subway box.

## Inference

Some exact station architectural details were inferred from corridor history and surviving station descriptions rather than reproduced from measured drawings. The goal of this pass is historically distinct corridor identity, not a survey-grade reconstruction.
